# Instructions to run the application

**Backend**

- Open the backend folder: 
- `cd backend`
- Run yarn to install the dependencies:
- `yarn`
- Start the server:
- `yarn dev:server`

**Frontend**

- Open the frontend folder: 
- `cd frontend`
- Run yarn to install the dependencies:
- `yarn`
- Start the server:
- `yarn start`
- Wait for the browser to open
